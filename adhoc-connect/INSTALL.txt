FIRST TIME INSTALLATION PROCEDURE:

Installation assumes a system running Red Hat Enterprise Linux 6.x in
enforced mode with the dhcp package installed.


Create the installation directory:

mkdir -p /cdg/adhoc/
restorecon -FR /cdg
cd /cdg/adhoc/

Expand and configure the distribution archive:

tar -xof adhoc-connect-x.y.z.tar
cd adhoc-connect-x.y.z
./install-upgrade.sh finish

---------------------------------------------------------------------------

UPGRADE PROCEDURE:

cd /cdg/adhoc/adhoc-connect

./install-upgrade.sh

Ths script will prompt for installation of the latest version if that is not already installed.
